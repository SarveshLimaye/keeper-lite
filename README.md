# Quick Notes
## Description
Useful and awesome note-taking app built using react js

## Tech-Stacks Used
 - React.js
 - Material-ui
 - Firebase
 - CSS
 - HTML

## Setting up Locally 

1. Fork the repo
```
2. git clone "https://github.com/[Your_GithubUserName]/keeper-lite.git"
```
```
3. cd keeper-lite
``` 
```
4. npm install 
``` 
```
5. npm start
```
